Projet BDE

Projet BDE, site web d'informations pour étudiants

Réalisation d’un site Web pour le bureau des étudiants. Le site web du BDE va permettre d’informer les étudiants de l’ORT sur les divers évènements organisés par le BDE, de présenter les actualités concernant le BDE mais également de présenter la liste du BDE. Utilisation du CMS WordPress pour faciliter la maintenance du site. Ce projet était un travail à faire en binôme.

*CMS (Content Management System) WordPress : création de sites web en ligne, open source.
